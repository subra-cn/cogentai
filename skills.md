## Frontend Development

### HTML & Markup
- **HTML5 Semantic Elements** — header, nav, main, section, article, aside, footer
- **Accessible Markup** — ARIA attributes, landmark roles, focus management, skip links
- **Structured Data** — JSON-LD, Schema.org, WebSite and Organization schemas
- **Open Graph / Meta** — Social sharing optimization, Twitter Cards, meta descriptions
- **Favicon & PWA** — SVG favicons, web app manifest, touch icons

### CSS & Styling
- **CSS Custom Properties** — Design tokens, theme variables, dynamic theming
- **CSS Grid** — Two-dimensional layouts, template areas, responsive grid patterns
- **Flexbox** — One-dimensional alignment, spacing, wrapping, ordering
- **CSS Animations** — @keyframes, transitions, scroll-triggered animations, performance-safe transforms
- **Media Queries** — Responsive breakpoints, prefers-reduced-motion, prefers-color-scheme
- **Clamp & Fluid Typography** — Viewport-relative sizing, fluid scaling, readability optimization
- **BEM Methodology** — Block-Element-Modifier naming convention, component scoping
- **Pseudo-elements** — Decorative gradients, overlays, indicators, visual effects

### JavaScript
- **ES6+ Syntax** — Arrow functions, destructuring, template literals, modules, async/await
- **DOM Manipulation** — QuerySelector, classList, event delegation, DOM traversal
- **IntersectionObserver** — Scroll-triggered animations, lazy loading, visibility tracking
- **Event Handling** — Click, scroll, resize, keyboard events, passive listeners
- **Progressive Enhancement** — Core functionality without JS, enhanced with JS
- **Form Handling** — Validation, submission, state management, success/error feedback
- **Smooth Scrolling** — Anchor navigation, scroll-into-view, scroll-padding
